# AI 활용 가이드

AI 도구를 효과적으로 사용하기 위한 실전 가이드를 정리하는 프로젝트입니다.

## 목차

- [내가 아는 것 활용하기](docs/known-knowns.md) - 이미 아는 영역에서 AI를 속도 부스터로 쓰기
- [정확한 답변 얻는 법](docs/getting-accurate-answers.md) - 모르는 것을 물어볼 때 hallucination 줄이기
- [모르는 것을 발견하는 법](docs/unknown-unknowns.md) - 질문조차 떠올리지 못하는 영역 탐색하기
- [프롬프트 엔지니어링](docs/prompt-engineering.md) - 좋은 프롬프트를 작성하는 방법
- [코딩 어시스턴트 활용](docs/coding-assistant.md) - AI 코딩 도구 활용 팁
- [워크플로우 통합](docs/workflow.md) - 일상 개발 워크플로우에 AI를 녹여내는 방법

## 다루는 주제

### 프롬프트 엔지니어링
- 명확하고 구체적인 지시 작성법
- 컨텍스트 제공 전략
- Few-shot, Chain-of-thought 등 기법 활용

### 코딩 어시스턴트
- Claude Code, Cursor, GitHub Copilot 등 도구별 특징과 활용법
- 코드 리뷰, 디버깅, 리팩토링에 AI 활용하기
- AI와 효과적으로 페어 프로그래밍하는 방법

### 워크플로우
- 문서 작성, 커밋 메시지, PR 리뷰 자동화
- AI 기반 검색과 학습 전략
- 프로젝트 규모별 AI 활용 패턴
